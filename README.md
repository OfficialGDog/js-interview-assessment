# js-interview-assessment

## About
A simple currency conversion tool

**Features**:
* Users can input an amount for the “from” currency.
* Fetch the converted value from the https://api.currencybeacon.com/v1/convert API.
* Populate the “to” value based on the value returned from the API.

## Getting Started
If you would like to run the app yourself you can follow these steps:

1\. Clone the project, to do this run the following commands
```bash
git clone https://github.com/OfficialGDog/js-interview-assessment
cd js-interview-assessment
npm install
```

2\. Start the application
```bash
npm run dev
```

3\. *Optional* run the unit tests
```bash
npm test
```
